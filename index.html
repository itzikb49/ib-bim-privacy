Privacy Policy for IB-BIM Applications

Effective Date: November 2024 | Last Updated: February 2026

Quick Summary: IB-BIM Applications do not collect or transmit any personal or project data. Minimal local usage statistics are stored on your computer only. Limited network connections are made solely for license validation and update checking — no user data is transmitted.

Scope

This Privacy Policy applies to all IB-BIM Revit Add-ins distributed via the Autodesk App Store, including:

Paid Applications:
  • Visibility Manager
  • Type Manager Pro
Free Applications:
  • Get Linked IDs

Information We Collect

IB-BIM Applications do not collect, upload, or transmit any personal information, project data, or Revit model content.

Local Data Storage

Stored locally on your computer only:

Paid applications (Visibility Manager, Type Manager Pro) store minimal usage statistics in a local JSON file at:

%LocalAppData%\IB-BIM\[AppName]\analytics.json

This file contains:

DataPurpose
First and last use dateDetermine when to show rating prompt
Session and operation countDetermine when to show rating prompt
Revit versionInternal compatibility reference
Application versionInternal version tracking
Rating prompt statusAvoid repeated rating requests

Important:

  • This data is never transmitted over the network
  • This data is never shared with any third party
  • You can delete this file at any time without affecting the application
  • No personal information, project data, or model content is stored

Network Connections

Limited network access — no user data transmitted:

IB-BIM paid applications make the following network connections:

ConnectionDestinationPurposeData Sent
License Validation Autodesk Entitlement API
(apps.autodesk.com)		 Verify active subscription or trial Autodesk User ID and App ID only
Update Check GitHub
(raw.githubusercontent.com)		 Check for new app versions (every 7 days) None — read-only download of a public JSON file

Free applications (Get Linked IDs) do not make any network connections and operate entirely offline.

Third-Party Services

IB-BIM Applications do not connect to or share data with any third-party analytics, advertising, marketing, or telemetry platforms. The only external connections are to Autodesk (license validation) and GitHub (update check) as described above.

Project Data & Revit Models

IB-BIM Applications:

  • Operate entirely within your local Revit environment
  • Do not access, copy, or transmit any project data or model content
  • Do not store any Revit element data outside of the current session
  • Exported Excel files are saved locally to a location you choose

Children's Privacy

IB-BIM Applications are intended for professional use and do not knowingly collect information from anyone, including children under 13 years of age.

Security

Since IB-BIM Applications do not collect or transmit personal data, there are no privacy or data breach risks. All operations are performed within Revit's secure environment. Local analytics files contain no personally identifiable information.

Your Rights

  • You can delete local analytics data at any time by removing the files in %LocalAppData%\IB-BIM\
  • No personal data is collected, so there is no data to access, modify, or request deletion of
  • No consent is required for data processing as no personal data is processed

Changes to This Policy

This Privacy Policy may be updated occasionally to reflect changes in our applications. Any updates will be reflected in the "Last Updated" date above.

Compliance

IB-BIM Applications comply with:

  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
  • Other applicable privacy regulations

Autodesk Platform

All IB-BIM Add-ins are distributed via the Autodesk App Store and comply with Autodesk's privacy and data security standards. Learn more at Autodesk's Privacy Statement.
